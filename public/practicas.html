Todas las practicas deben estar en el repositorio con su respectivo Makefile para que sean tomadas en cuenta. Practica

0.1 FizzBuzz

Lenguaje de implementacion: C -Dada una entrada de n numeros, imprimir los numeros linea a linea, teniendo en cuenta la siguiente condicion: Si n es divisible entre 3, imprimir Fizz Si n es divisible entre 5, imprimir Buzz Si n es divisible entre 3 y 5, imprimir FizzBuzz

0.2 2K38Y

Lenguaje de implementacion: C -Imprimir la fecha máxima en la cual la arquitectura de 32bits soportara implementaciones de ctime, es decir, la fecha en la cual sistemas que usan esta implementación de medicion de tiempo colapsaran. Para realizar esta practica es necesario emplear la biblioteca ctime.h, asi como limits.h, limits.h contiene los valores maximos que puede almacenar una variable, mientras que ctime contiene funciones de impresion de fechas. Algunas de estas funciones reciben un numero como parametro, que representa los segundos transcurridos desde el 1 de Enero de 1970, si se tiene el valor maximo que puede tomar un entero de 32bits, se podrá saber la fecha y la hora exacta de este evento.
